# FRONTEND-INTERN
2022 FRONTEND-INTERN-HOMEWORK

1. Clone 專案
2. 執行 ```npm install```
3. 執行 ```npm start```

目標 : 

- 串接 Github REST API，並使用 REACT 實作一個瀏覽單一使用者 Github Repository 的網站。
- 完成作業後，上傳到 Github，繳交連結。

基本要求 :
- 實作 Repository 列表
- [x] 必須是 `users/{username}/repos`
